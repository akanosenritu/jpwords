<ESP>する (hacer) es el verbo más bacio en japonés y tiene su propios significados. Además する se puede usar para hacer una palabra compuesta con sustantivos de acción. Por ejemplo: </ESP>

* <Ruby t={"べんきょう"}>勉強</Ruby> (study) + する = 勉強する (to study)
* <Ruby t={"あんない"}>案内</Ruby> (guide) + する = 案内する (to guide)
* サッカー (football) + する = サッカーする (to play football)

Most of the time these nouns can be used as the direct object of する.
* 勉強をする = 勉強する
* 案内をする = 案内する
* サッカーをする = サッカーする
